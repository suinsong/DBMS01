# DBMS 기본 명령체계

## SQL(Structure Query Language)

명령             대상                       조건
create    DB, Table       x        구조 만들기
insert    Table           x        Record 단위 추가
select    Table         어떤,무엇        Data List열람
updata    Table         어떤,무엇        Record 단위 수정
delect    Table         어떤,무엇       Record 단위 삭제
